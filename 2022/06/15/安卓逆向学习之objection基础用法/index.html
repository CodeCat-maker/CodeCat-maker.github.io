安卓逆向学习之objection基础用法

0x1 介绍&安装 Objection

objection 是基于 frida 的命令行 hook 工具, 可以让你不写代码, 敲几句命令就可以对 java 函数的高颗粒度 hook, 还支持 RPC 调用

目前只支持 Java 层的 hook, 但是 objection 有提供插件接口, 可以自己写 frida 脚本去定义接口

比如葫芦娃大佬的脱壳插件, 实名推荐: FRIDA-DEXDump

官方仓库: objection

  • 安装前置条件
  • 安装命令

安装完成后, 直接输入objection, 就可以看到食用方法了

0x2 简单使用一下

  • 使用前几个使用 tips
  • 启动 Frida-server 并转发端口
  • 附加需要调试的 app, 进入交互界面
  • 可以使用该 env 命令枚举与所讨论的应用程序相关的其他有趣目录: env
  • 我们可以使用以下 file download 命令从远程文件系统中下载文件:
    file download [file] [outfile]

  • 可以列出 app 具有的所有 avtivity: android hooking list activities

  • 启动指定 avtivity: android intent launch_activity [class_activity]

  • RPC 调用命令: curl -s "http://127.0.0.1:8888/rpc/invoke/androidHookingListActivities"
